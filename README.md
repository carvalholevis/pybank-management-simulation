# 💰 PyBank Management Simulation

A command-line banking system simulation built with Python, focused on
core financial operations and clean logical structure.

This project was developed as part of the Python Fundamentals program by
Digital Innovation One (DIO), with an emphasis on building strong
programming foundations and problem-solving skills.

------------------------------------------------------------------------

### 🚀 Project Overview

PyBank is a simple yet effective simulation of a banking system that
runs in the terminal (CLI). It demonstrates how real-world business
rules can be translated into code using fundamental Python concepts.

This project highlights the ability to: - Translate requirements into
functional logic - Structure a program clearly and maintainably - Handle
user interaction through terminal interfaces - Apply core programming
concepts in a practical scenario

------------------------------------------------------------------------

### ⚙️ Features

-   💵 Deposit functionality
-   💸 Withdrawal with constraints (limits and rules)
-   📊 Real-time balance tracking
-   📜 Transaction history
-   🧠 Interactive menu system (CLI-based)

------------------------------------------------------------------------

### 🛠️ Tech Stack

-   Python 3
-   Command Line Interface (CLI)
-   Core programming constructs (conditionals, loops, functions)

------------------------------------------------------------------------

### 📂 How to Run

1. Clone the repository

    git clone https://github.com/carvalholevis/pybank-management-simulation.git

2. Navigate into the project folder

    cd pybank-management-simulation

3. Run the application

    python main.py

  Make sure you have Python 3 installed.
